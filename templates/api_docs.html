Base URL

Authentication

API access requires an API key which should be sent via the Authorization header: 

Authorization: Bearer YOUR_API_KEY

Contact our support team to request an API key.

Endpoints

1. Get Forecast

POST /forecast

Request a forecast for a given metal, model, and horizon.

Request Body (JSON):
{
  "metal": "gold" | "silver",
  "model": "ARIMA" | "RandomForest" | "LSTM",
  "horizon": integer (1-30)
}
Response (JSON):
{
  "status": "success",
  "metal": "gold",
  "model": "ARIMA",
  "horizon": 7,
  "forecast": [1800.25, 1798.50, 1795.75, 1792.80, 1790.10, 1788.50, 1786.75]
}
Example Curl:
curl -X POST https://yourdomain.com/api/v1/forecast \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{"metal":"gold","model":"ARIMA","horizon":7}'

Error Handling

Errors will be returned in the following format:

{
  "status": "error",
  "message": "Description of the error"
}
