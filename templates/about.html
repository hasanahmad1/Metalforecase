About This Project

This website offers a powerful platform to forecast the prices of gold and silver using advanced machine learning and statistical methods:

  • ARIMA: A classic time series forecasting model widely used for analyzing and predicting trends.
  • Random Forest: An ensemble learning method combining multiple decision trees for robust regression results.
  • LSTM: A type of recurrent neural network designed to capture long-term dependencies in sequential data for accurate predictions.

Users can select the metal, forecasting model, and horizon to obtain accurate, real-time price predictions supported by clear visualizations.
